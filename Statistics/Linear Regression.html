What is Linear Regression?

(Simple) Linear Regression is a statistical model which allows us to predict the relationship between the dependent and independent variables by assuming there exists a straight line which connects the known data in a optimal way. There are many possibilities for an optimal way to construct such a straight line. One commonly used method in linear regression is known as the method of least squares first described by Carl Friedrich Gauss.

Simple Linear Regression

This is the simplest case of linear regression where we only consider one independent and one dependent variable. Recall that such equations can be modelled using the equation $$y_i=\beta_0 + \beta_1 x_i + \epsilon_i$$ where each \(i\) is a counter for each observation.

This model requires us to estimate the slope and intercept of the straight line. The slope represents a change in the depedent variable for one unit change in the indepedent variale while the intercept represents the predicted value of the depedent variable when the independent variable is zero.

Minimizing the Cost Function

Random Error

Using the philosophy of the method of least squares, our goal is to find the best values of \(\beta_0\) and \(\beta_1\). That is, we want to minimize the sum of the difference between the actual data points that we observed and the predicted points on the line. This is known as the random error, and can be obtained from the previous equation. $$\epsilon_i = y_i - (\beta_0 + \beta_1 x_i)$$

Minimizing the MSE

The Mean Squared Error (MSE) measures the average of the squared difference between the actual data points that we observed and the predicted points on the line. $$MSE = \frac{1}{N} \sum_{i=1}^n (y_i - (\beta_0 + \beta_1 x_i))^2$$

To minimize such a function, we should first take partial derivatives of the MSE with respect to \(\beta_0\) and \(\beta_1\), and set them to \(0\). Hence, we get the normal equations $$\displaystyle{\frac{\partial MSE}{\partial \beta_0} = \frac{2}{n} \sum_{i=1}^n (y_i - (\beta_0 + \beta_1 x_i))} = 0 \tag{1}$$ and $$\displaystyle{\frac{\partial MSE}{\partial \beta_1} = \frac{2}{n} \sum_{i=1}^n (y_i - (\beta_0 + \beta_1 x_i))x_i} = 0 \tag{2}$$
